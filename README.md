# tj-cli

天鸡cli，一个使用简单，不限模板类型的脚手架。

## 安装

- NodeJS 10+： <https://nodejs.org/en/>
- yarn：<https://yarn.bootcss.com/>

```bash
yarn global add @lijg/tj
```

## 创建项目

```bash
cd ~/project # 存放项目的路径
tj create <project name> # <project name> 替换为项目名称
```
